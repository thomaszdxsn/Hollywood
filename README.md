# Hollywood

## Usage

* python
    
    1. install python3.7

        `brew install python3`

    2. install pipenv

        `python3.7 -m pip install pipenv`

    3. install dependencies

        `pipenv install`

    4. run python server

         `cd src`
        

        `pipenv run python manage.py runserver`

* node

    1. install dependencies

        `npm install`

    2. run dev

        `npm run dev`

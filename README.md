# Hollywood

## Usage

* python
    
    install python3.7

        `brew install python3`

    install pipenv

        `python3.7 -m pip install pipenv`

    install dependencies

        `pipenv install`

    run python server

         `cd src`
        

        `pipenv run python manage.py runserver`

* node

    install dependencies

        `npm install`

    run dev

        `npm run dev`
